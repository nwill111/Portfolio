Airborne

  • Category: Personal Project
  • Project date: June 2022 - July 2022
  • Project URL: Github

My first unity project, a platforming/fighting game

Airborne is a fighting/platforming game that I developed in 3 weeks, starting from the ground up with zero prior experience in unity. I wanted to challange myself and see how much I could learn about game development and how complex of a game I could make in the timeframe that was provided to me. The game includes features such as attack combos, a complex movement system, and intellegent enemies.

The game uses 20+ custom written C# scripts.
